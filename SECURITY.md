# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ICS Ninja Scanner, please report it responsibly.

**DO NOT** open a public GitHub issue for security vulnerabilities.

### How to Report

Email: **security@mottasec.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response Timeline

- **Acknowledgment:** Within 48 hours
- **Initial assessment:** Within 1 week
- **Fix or mitigation:** Depends on severity

### Scope

This policy covers vulnerabilities in the ICS Ninja Scanner tool itself. If you find a vulnerability in a target system while using this tool, please report it to the system owner through appropriate channels.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅        |
| < 1.0   | ❌        |

## Responsible Use

This tool is designed for authorized security assessments only. Using it against systems without authorization is illegal and unethical. MottaSec is not responsible for misuse of this tool.
